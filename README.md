# Sheikh El 3ashabin (شيخ العشابين) - E-Commerce Platform

Complete e-commerce solution for Sheikh El 3ashabin herbal store, built with Laravel 10 + Vue.js 3.

## 🌿 About

Sheikh El 3ashabin is a traditional herbal and natural products store bringing the finest herbs and natural remedies to customers.

## ✨ Features

- Product catalog management
- Shopping cart & checkout
- Order management system
- Multiple payment gateways (Cash on Delivery, Credit Card, InstaPay, Mada)
- Inventory tracking
- Admin dashboard
- Responsive design
- Arabic & English support

## 🛠️ Tech Stack

- **Backend:** Laravel 10, PHP 8.2
- **Frontend:** Vue.js 3, Tailwind CSS
- **Database:** MySQL 8.0
- **Server:** Nginx, Ubuntu 22.04
